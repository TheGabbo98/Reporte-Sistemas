# Reporte de Sistemas

Sistema interno de registro de reportes técnicos.

## Acceso
El sitio está deployado en Netlify. Ingresá con tu usuario y contraseña asignados.

## Estructura
- `index.html` — aplicación completa (HTML + CSS + JS en un solo archivo)

## Deploy
Este repositorio está conectado a Netlify. Cualquier push a `main` actualiza el sitio automáticamente.

## Tecnologías
- HTML/CSS/JS puro (sin frameworks, sin dependencias externas)
- Chart.js para gráficos
- Google Apps Script como backend
- SHA-256 (Web Crypto API) para contraseñas
